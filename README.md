# Фитнес-лендинг Кати Усмановой — копия 3 экранов

Аккуратная адаптивная копия трёх экранов лендинга usmanovafit.gymteam.ru/mainpage: Hero, блок доверия «О Кате», каталог «Тренировки дома».

## Что внутри

Повторены три экрана оригинального сайта:

1. **Hero / первый экран** — крупный заголовок, подзаголовок, описание, розовая CTA-кнопка, справа фото Кати с гантелью поверх розово-сиреневой blob-формы.
2. **Блок доверия** — заголовок про чемпионку, фото Кати слева, справа список из 5 преимуществ с иконками в розовых кругах.
3. **Каталог «Тренировки дома»** — 6 карточек программ (Метод Усмановой, Стройности, Упругая попа 1.0/2.0, Плоский живот, Жиросжигающий) с изображениями, метками, описаниями и кнопками «Подробнее».

Дополнительно реализован интерактив:
- плавный скролл по CTA-кнопке Hero к секции программ;
- модальное окно с формой заявки (Имя, Телефон, Цель, согласие);
- валидация полей и вывод заявки в console.log;
- сообщение об успешной отправке;
- cookie-плашка с сохранением согласия в localStorage.

## Стек

- Чистый HTML / CSS / JavaScript (без сборщиков и фреймворков).
- Шрифт Manrope из Google Fonts (ближайший бесплатный аналог платного Gilroy из оригинала).
- Все изображения подключены локально из папки assets.

## Как запустить локально

Так как это статичный сайт, достаточно открыть index.html в браузере.
Для корректной работы шрифтов и относительных путей удобно поднять простой локальный сервер:

```bash
# Python 3
python -m http.server 5500

# или Node
npx serve .
```

Затем открыть http://localhost:5500 (или предложенный порт).

## Проверенные брейкпоинты

- 360px / 375px / 390px / 430px — мобильная раскладка, всё в одну колонку.
- 768px — планшет, программы в 2 колонки.
- 1024px / 1366px / 1440px — десктоп, 3 колонки программ, Hero в 2 колонки.

На всех размерах проверено:
- нет горизонтального скролла;
- декоративные blob-формы не вылезают за пределы экрана (overflow: hidden у секций);
- фото в карточках не растягивается некрасиво;
- модалка помещается в экран;
- форма и cookie-плашка работают.

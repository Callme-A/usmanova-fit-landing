# Фитнес-лендинг Кати Усмановой — копия экранов

Адаптивная копия экранов лендинга `usmanovafit.gymteam.ru/mainpage`: Hero, блок доверия «О Кате», промежуточный CTA «Кате доверяют миллионы», каталог «Тренировки дома». Собрано как статика на HTML/CSS/JS и визуально приближено к оригинальному промо-лендингу.

## Что внутри

Повторены и доработаны экраны оригинального сайта:

1. **Hero / первый экран** — крупный плотный заголовок, подзаголовок, описание, розовая CTA-кнопка, справа большое фото Кати с гантелью, слабая розовая подложка.
2. **Блок доверия** — крупный заголовок про чемпионку, список регалий с компактными иконками, PNG-коллаж Кати с прозрачным фоном поверх мягкой розовой плашки.
3. **Промежуточный CTA «Кате доверяют миллионы»** — розовая полноцветная секция с социальным доказательством и кнопкой «Вернуть форму».
4. **Каталог «Тренировки дома»** — 6 карточек программ (Метод Усмановой, Стройности, Упругая попа 1.0/2.0, Плоский живот, Жиросжигающий) с крупными изображениями, метками, описаниями и кнопками «Подробнее».

Интерактив:
- плавный скролл по CTA-кнопкам Hero и social-proof к секции программ;
- модальное окно с формой заявки (Имя, Телефон, Цель, согласие);
- валидация полей и вывод заявки в `console.log`;
- **отправка заявок на почту `sasha.www.ru78@gmail.com`** через сервис FormSubmit.co (без бэкенда);
- защита от двойной отправки и полный сброс состояния формы при повторном открытии;
- сообщение об успешной отправке;
- cookie-плашка с сохранением согласия в `localStorage`.

## Как активировать отправку писем на почту

Отправка работает через бесплатный сервис [FormSubmit.co](https://formsubmit.co), который не требует ключей API и регистрации.

1. При первой отправке заявки FormSubmit пришлёт на `sasha.www.ru78@gmail.com` письмо с подтверждением.
2. Откройте это письмо и нажмите ссылку подтверждения.
3. После этого все последующие заявки будут приходить на почту сразу.

Если письмо подтверждения не приходит — проверьте папку «Спам».

## Как запустить локально

Достаточно открыть `index.html` в браузере или поднять простой локальный сервер:

```bash
python -m http.server 5500
# затем открыть http://localhost:5500
```

## Проверенные брейкпоинты

- 360px / 375px / 390px / 430px — мобильная раскладка.
- 768px — планшет, программы в 2 колонки.
- 1024px / 1366px / 1440px — десктоп, 3 колонки программ.

На всех размерах проверено: нет горизонтального скролла, фото не вылезает за экран, модалка открывается и закрывается, форма валидируется и отправляется, cookie-плашка работает.
